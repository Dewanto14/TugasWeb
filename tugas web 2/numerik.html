5. Numeric


Keyboard selanjutnya yang juga banyak digunakan sebagai alat input komputer adalah keyboard numerik. Karena terdiri dari angka-angka keyboard ini dikhususkan untuk melakukan perhitungan di Microsoft Excel sehingga tidak bisa digunakan di Microsoft word. Terdapat pengecualian terkait penggunaan Microsoft word dengan menggunakan keyboard numeric. Asalkan data yang dimasukkan berupa angka, maka bisa diolah. Sedangkan, bila berupa huruf tidak bisa karena keyboard ini hanya terdiri dari menu serupa calculator. Banyaknya jenis keyboard pc membuat anda terkadang bingung harus menggunakan yang mana. Ketika ingin melakukan pekerjaan dengan cepat, akan lebih baik bila anda memahami kelebihan dan kekurangan dari masing-masing keyboard yang disambungkan ke pc.
